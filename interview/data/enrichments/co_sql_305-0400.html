Pinterest invented visual search ("Lens"). The CTR comparison: visual queries vs text queries reveals whether visual search is matching user intent better.

2. The SQL — CTR per query type

SELECT query_type,
  COUNT(*) AS searches,
  SUM(clicked) AS clicks,
  ROUND(100.0 * SUM(clicked) / NULLIF(COUNT(*), 0), 2) AS ctr_pct
FROM searches GROUP BY query_type
ORDER BY ctr_pct DESC;

Why visual wins — concrete intent

"Find shoes like this picture" is a precise query. "Find shoes" is broad. Visual search matches "this exact thing" — narrower, higher-relevance results.

The discovery-vs-shopping split

Visual search dominates shopping (price-comparison). Textual dominates discovery (browsing inspiration). The query above lumps them; production splits by query intent.

Mobile-only signal

Visual search is mostly mobile (camera-based). Comparing CTR across query types confounds device. Slice by device type for fair comparison.

The Try-On / AR cousin

Pinterest's AR Try-On extends visual search — see the product on yourself. Saves at 2-3x rate of plain visual search. Production tracks Try-On separately.

Senior signal: visual CTR exceeds textual due to specificity, split by discovery vs shopping intent, control for device, and surface Try-On as the visual-search extension.
