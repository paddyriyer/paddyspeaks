App Store Search is one of Apple's hidden ML systems. CTR per query term measures search-relevance quality. The query is straightforward GROUP BY + sample-size HAVING.

1. The source data — app_searches

search_idquery_termapp_idpositionclicked
S_001"weather"APP_0111

2. The SQL — per-query-term CTR

SELECT query_term,
  COUNT(*) AS impressions,
  SUM(clicked) AS clicks,
  ROUND(100.0 * SUM(clicked) / NULLIF(COUNT(*), 0), 2) AS ctr_pct
FROM app_searches
GROUP BY query_term
HAVING COUNT(*) >= 3
ORDER BY ctr_pct DESC;

3. Strategic explanations

Branded vs generic queries

Searches for "Spotify" have CTR ≈ 95% (user knew what they wanted). Searches for "music" have CTR ≈ 30% (multiple apps compete). Mixing them gives a meaningless average. Always classify:

CASE WHEN EXISTS (SELECT 1 FROM apps a WHERE LOWER(a.name) LIKE LOWER(query_term) || '%')
     THEN 'branded' ELSE 'generic' END AS query_type
Position confound

CTR depends on which apps appear in slots 1-5. A query with great results at position 1 has high CTR; the same query with mediocre results has low CTR — same query, different supply. Slice by query × top_app_quality.

The "no-click → search-again" pattern

Low-CTR queries often signal poor relevance. Combined with reformulation rate (user searches again immediately after), the diagnostic is precise:

... user reformulated within 30 seconds = sign that this query failed
Promo vs editorial vs algorithm slot

App Store has three slot types: paid promotion, editorial pick, algorithm. Each has different CTR characteristics. Comparing them is unfair. Production splits.

The search-stickiness signal

Per-query CTR is search-engine quality. Per-user retention after first install is product quality. Production tracks both: high CTR + low retention = "Apple's search is great, but the apps don't deliver".

Senior signal: classify branded vs generic, control for position/slot quality, link low-CTR to immediate reformulation, distinguish promo/editorial/algorithm slots. The CTR-vs-retention complementary metric is the over-the-bar honest framing.
