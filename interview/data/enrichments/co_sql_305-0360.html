Stripe's ARR (Annual Recurring Revenue) chart slices subscribers by signup-month cohort and tracks how many remain active at each subsequent month. The query is the canonical cohort matrix using FIRST_VALUE for the denominator.

2. The SQL — month-N retention with FIRST_VALUE

WITH joined AS (
  SELECT signup_month,
    CAST((julianday(active_month || '-01') - julianday(signup_month || '-01')) / 30 AS INTEGER) AS month_n,
    customer_id
  FROM subscriptions
)
SELECT signup_month, month_n,
  COUNT(DISTINCT customer_id) AS active_customers,
  ROUND(100.0 * COUNT(DISTINCT customer_id) /
    FIRST_VALUE(COUNT(DISTINCT customer_id)) OVER (PARTITION BY signup_month ORDER BY month_n), 1) AS retention_pct
FROM joined
WHERE month_n BETWEEN 0 AND 12
GROUP BY signup_month, month_n
ORDER BY signup_month, month_n;

3. Strategic explanations

FIRST_VALUE for the cohort baseline

The denominator for retention% is the cohort's month-0 size. FIRST_VALUE(...) OVER (PARTITION BY signup_month ORDER BY month_n) grabs the count from month_n=0 and replicates it across rows of the same cohort — one-pass instead of a self-join.

The month_n calculation

julianday(active_month || '-01') - julianday(signup_month || '-01') is the day-difference. Dividing by 30 approximates months (rough but adequate for cohort buckets).

The in-flight cohort hide

Cohorts < 12 months old don't have full retention data — month-12 cells are NULL. Either hide them with WHERE julianday(signup_month) <= julianday('now') - INTERVAL '13 months' OR render them with explicit NULL.

Net vs gross retention
metricformula
Gross retentioncustomers retained / cohort size
Net dollar retention (NDR)(retained MRR - churned MRR + expansion MRR) / cohort MRR

NDR can exceed 100% (expansion outweighs churn). The query above is gross. SaaS dashboards focus on NDR.

The cohort comparison

The PM-actionable view is side-by-side: this month's cohort vs last quarter's. Use a pivot to compare:

MAX(CASE WHEN signup_month = '2024-01' THEN retention_pct END) AS cohort_jan
MAX(CASE WHEN signup_month = '2024-04' THEN retention_pct END) AS cohort_apr
Senior signal: use FIRST_VALUE as the cleanest cohort denominator, hide in-flight cohorts, contrast gross retention with NDR, diagnose curve shape (cliff at M12 = annual contracts), and pivot for cohort-vs-cohort comparison.
