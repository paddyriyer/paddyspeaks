Returns are Amazon's #1 fulfillment cost driver. The category-level return rate is the input to Vendor Central's quality scoring. The query LEFT JOINs orders to returns and computes per-category ratio.

1. The source data — orders + returns

2. The SQL — LEFT JOIN + ratio + sample-size HAVING

SELECT o.category,
  COUNT(DISTINCT o.order_id) AS orders,
  COUNT(DISTINCT r.return_id) AS returns,
  ROUND(100.0 * COUNT(DISTINCT r.return_id) / NULLIF(COUNT(DISTINCT o.order_id), 0), 1) AS return_rate_pct
FROM orders o
LEFT JOIN returns r ON r.order_id = o.order_id
GROUP BY o.category
HAVING COUNT(DISTINCT o.order_id) >= 5
ORDER BY return_rate_pct DESC;

3. Strategic explanations

The denominator pitfall

"Return rate = returns / orders" — but which orders? Only delivered orders can be returned. Including in-flight orders deflates the rate. Production filters WHERE o.delivered_at IS NOT NULL.

The "fulfillment-window" rule

Amazon's return window is 30 days. A return filed 91 days after delivery shouldn't count against last month's return rate. Add a temporal predicate:

LEFT JOIN returns r ON r.order_id = o.order_id
  AND julianday(r.refunded_at) - julianday(o.delivered_at) <= 30
The category-quality vs category-policy split

Apparel has structurally high return rates (size mismatch). Books have structurally low. Don't flag Apparel as "broken" — compare each category to its baseline:

... return_rate_pct - AVG(return_rate_pct) OVER () AS deviation_from_avg
Reason mix — the diagnostic

Return rate alone is the symptom. The PM-actionable view groups by reason:

GROUP BY o.category, r.reason

"Damaged" is fulfillment-quality; "wrong_item" is picking-error; "not_as_described" is listing-quality. Three different fixes.

Per-seller drill-down

Aggregating to category hides individual bad actors. The actionable view is per-seller within category — top-N return-rate sellers within Apparel are the Vendor Central audit list.

Senior signal: filter to delivered orders for honest denominator, add the 30-day return-window predicate, compare to category baseline (not absolute), and split reason for actionable diagnosis.
