Apple's iPhone upgrade cycle is the company's central revenue rhythm. The query measures months between consecutive purchases per customer — the input to demand forecasting + trade-in pricing.

1. The source data — device_purchases

2. The SQL — LAG-based gap detection

WITH gaps AS (
  SELECT customer_id, device_model, purchased_at,
    LAG(purchased_at) OVER (PARTITION BY customer_id ORDER BY purchased_at) AS prev_purchase
  FROM device_purchases
)
SELECT customer_id, device_model, purchased_at,
  CAST((julianday(purchased_at) - julianday(prev_purchase)) / 30.0 AS INTEGER) AS months_since_last
FROM gaps
WHERE prev_purchase IS NOT NULL
ORDER BY months_since_last DESC;

3. Strategic explanations

Cycle length distribution — what it reveals
months_since_lastsegment
1-12premium-cycle (every iPhone launch)
13-24two-year cycle (carrier contracts)
25-36standard upgrade
37+late-cycle (price-sensitive)

Apple's installed-base growth model breaks customers into these buckets. The 25-36 bucket is the largest; that's the demand-forecasting input.

Device-line-cross signal

iPhone-only customers are different from iPhone+iPad+Mac. Multi-device customers have shorter cycle (more devices to upgrade more often). Group:

GROUP BY customer_id, CASE WHEN device_model LIKE 'iPhone%' THEN 'phone' ELSE 'other' END
The "first purchase" exclusion

The first iPhone has no "previous" — the LAG is NULL. That's correctly excluded by the WHERE clause. New customers contribute to the SUM but not to the cycle metric.

Trade-in offset

Apple's trade-in program affects cycle: trade-in lowers effective new-device price, encouraging shorter cycles. Joining to trade_ins shows the lift.

Premium vs basic-tier cycle divergence

Pro models have shorter cycles (early-adopter audience); base models have longer (price-sensitive audience). Splitting by tier reveals divergent demand curves.

Senior signal: distribute customers into 12/24/36 month cycle buckets, distinguish iPhone-only from multi-device, surface trade-in's effect on cycle length, and split by Pro vs base tier. The 25-36 bucket as the demand-forecasting baseline is the senior frame.
