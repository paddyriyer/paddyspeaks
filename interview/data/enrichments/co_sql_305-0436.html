Per-store repeat-customer rate is the leading indicator of LTV. Stores with > 30% repeat-buyer rate retain customers; below 15% means each acquisition is a one-off.

2. The SQL — per-store repeat ratio

WITH per_customer AS (
  SELECT store_id, customer_id, COUNT(*) AS order_count
  FROM orders GROUP BY store_id, customer_id
)
SELECT store_id, COUNT(*) AS unique_customers,
  SUM(CASE WHEN order_count >= 2 THEN 1 ELSE 0 END) AS repeat_customers,
  ROUND(100.0 * SUM(CASE WHEN order_count >= 2 THEN 1 ELSE 0 END)
              / NULLIF(COUNT(*), 0), 1) AS repeat_rate_pct
FROM per_customer GROUP BY store_id HAVING COUNT(*) >= 3
ORDER BY repeat_rate_pct DESC;

3. Strategic explanations

The 30% benchmark

Above 30% = strong loyalty (subscription / replenishment products like beauty, supplements, food). 15-30% = average. Below 15% = one-off categories (furniture, weddings).

2-orders is the minimum bar

Some studies use 3+ orders for "loyal". 2 is the most permissive. Pin the threshold.

Time-window normalization

Lifetime repeat-rate vs 12-month repeat-rate are different. Production restricts to a window.

Category-baseline divergence

Apparel: 35% expected. Electronics: 12%. Comparing across categories without controlling unfairly judges electronics merchants. Stratify.

The CAC payback signal

Stores with high repeat rate can afford higher CAC (each customer worth more). The query feeds the marketing-budget allocator.

Senior signal: 30% benchmark, define "repeat" threshold explicitly, time-window for honest rate, stratify by category baseline, and connect to CAC-payback economics.
