The Room · First 90 seconds · scorecard

Four moves. Miss any one and the round drops.

  1. Refused to answer the question as asked. "Which is correct" → "neither; here is what each measures."
  2. Gave a specific time commitment. "45 minutes with a deliverable," not "I need some time."
  3. Ended with a recommendation, not a number. Finance gets a labelled recommendation, not a digit.
  4. Named what you don't know. "Depends on which definition the dashboard is computing" — humility is the signal.

It looks like a system-design round. It's an incident-response round with a design wrapper.

The Landmine · The label is the schema

"Revenue — September" is not one number.

At StreamCo it is four numbers in four systems. Someone labelled a dashboard column Revenue eighteen months ago. A hundred people built mental models on it. The label calcified. The engineering team inherited a political debt nobody spoke about.

The senior fix is not a query. Every dashboard at StreamCo gets a definition footer: which table, which attribution window, which revenue concept, which currency basis. New dashboards without the footer fail code review. Existing dashboards back-filled this quarter.

Rule: if a number has a label, the label is part of the schema.
